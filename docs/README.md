# Multiservicios JCD

Sitio web profesional para Multiservicios JCD - Soluciones integrales de servicios de reparación, mantenimiento y reforma.

## Servicios Ofrecidos

- **Electricidad**: Instalación, reparación y mantenimiento de sistemas eléctricos
- **Fontanería**: Servicios de agua, drenaje y sistemas hidráulicos
- **Plomería**: Reparación e instalación de sistemas de plomería
- **Reformas Generales**: Renovación integral de espacios
- **Reformas Eléctricas y Albañilería**: Modificaciones eléctricas y trabajos de construcción
- **Cerrajería**: Instalación y reparación de cerraduras

## Características

✅ Diseño responsivo (móvil, tablet, desktop)

## Cómo Personalizar

### Número de WhatsApp

En `index.html`, busca la línea del botón WhatsApp y reemplaza el número:

```html
<a href="https://wa.me/+34XXXXXXXXXX" class="whatsapp-button">
```

Reemplaza `+34XXXXXXXXXX` con tu número de WhatsApp (incluyendo el código de país).

### Colores

Puedes cambiar el color verde principal (`#00a86b`) en `css/styles.css` por tu color preferido.

## Visualización Local

1. Abre `index.html` en tu navegador
2. O usa un servidor local:
   - Con Python: `python -m http.server 8000`
   - Con Node.js: `npx http-server`

## Publicación en GitHub Pages

1. Copia el contenido a la carpeta `docs/`
2. Sube a GitHub
3. En Settings > Pages, selecciona Branch: `main` y Folder: `/docs`
